Локальное веб-приложение

Family Slideshow Maker

Фото + видео + музыка → слайдшоу с ручным кадрированием, форматами 16:9 / 4:3 / 1:1 / вертикальный и мягким зумом.

Файлы не загружаются на сервер Всё работает прямо в браузере. GitHub Pages нужен только чтобы открыть страницу.

1. Кадры и загрузка

Список кадров теперь сразу слева на одном уровне с предпросмотром. При просмотре переключай кадры стрелками ↑/↓ или ←/→. Shift+клик по чекбоксу/кадру выделяет пачку файлов.

Проект сохраняется локально в этом браузере вместе с медиафайлами. На сервер ничего не отправляется.

Видео сохраняют свою длительность. Оставшееся время музыки распределяется между фото.

3. Кадрирование и зум

Можно отдалять до 0.25×, чтобы влезал весь контент, или приближать для кинематографичного движения. Для проверки начала/конца выбранного кадра используй ползунок под предпросмотром.

Для видео длительность можно менять, но его звук в веб-MVP не смешивается — используется музыка.

4. Экспорт

Веб-экспорт остаётся WebM, но теперь файл дописывается с длительностью, чтобы нормально проматывался в плеерах. Для железобетонного MP4/H.264 под телевизор потом лучше добавить локальный FFmpeg-экспорт.
