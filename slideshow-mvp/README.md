# Family Slideshow Maker

Локальный веб-редактор для семейного слайдшоу.

## Что есть

- фото + видео + музыка;
- форматы 16:9, 4:3, 1:1, 9:16 и свой размер;
- порядок кадров, выделение пачкой, удаление пачки;
- стрелки ↑/↓ и ←/→ для переключения;
- Shift + клик / Shift + стрелки для диапазона;
- ручное кадрирование, зум, длительность;
- отдельная позиция кадра для начала и конца;
- колесо мыши над превью меняет зум, зажатая мышь двигает кадр;
- ползунок «Зум в начале/конце» сам переключает предпросмотр на нужную точку;
- проверка начала / середины / конца кадра;
- локальное сохранение проекта в браузере;
- прямой экспорт MP4-видео из браузера, если браузер поддерживает MP4 MediaRecorder;
- пресеты плавности экспорта: 1080p/24fps, максимум плавности 720p/24fps, качество 1080p/30fps;
- оптимизации браузерного рендера: сниженный битрейт по умолчанию, реже обновляется UI во время экспорта, кешируется размытый фон для фото;
- WebM только как быстрый черновик;
- AI авто-кадрирование лиц через локальный MediaPipe Face Detector;
- автоматический размытый фон для вертикальных/нестандартных фото.

## AI-функции

В блоке «Кадрирование» есть отдельная панель:

- **AI выбранный кадр** — найти лица и безопасно выставить кадрирование для текущего файла.
- **AI обработать все** — пройтись по всем фото/видео.
- **Фон для вертикальных** — включить «вписать + размытый фон» для вертикальных/нестандартных кадров.
- **Авто для новых** — новые файлы автоматически получают AI-кадрирование после загрузки.

Медиа не отправляются на сервер. Модель MediaPipe загружается в браузер и работает локально. Если CDN модели недоступен, приложение применяет базовое авто-кадрирование без распознавания лиц.
