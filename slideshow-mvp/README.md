# Family Slideshow Maker

Быстрое статическое веб-приложение для слайдшоу:

- загрузка фото;
- загрузка музыки;
- авто-длительность каждого фото по длине трека;
- ручной порядок фото;
- ручное кадрирование X/Y;
- стартовый и финальный zoom для простого Ken Burns эффекта;
- 16:9 предпросмотр;
- экспорт WebM в браузере через Canvas + MediaRecorder.

## Как запустить локально

Открой `index.html` в Chrome/Edge. Если браузер блокирует локальные файлы, запусти простой сервер:

```bash
python -m http.server 8080
```

И открой:

```text
http://localhost:8080
```

## Как разместить на GitHub Pages

1. Создай новый репозиторий на GitHub, например `slideshow-maker`.
2. Загрузи в него `index.html`, `styles.css`, `app.js`, `README.md`.
3. Открой Settings → Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder `/root`.
6. Через минуту сайт будет доступен по адресу GitHub Pages.

## Ограничение MVP

Веб-версия экспортирует `.webm`. Это нормально для браузера, но для телевизора чаще надёжнее `.mp4` / H.264. Для MP4 лучше добавить отдельный desktop/local экспорт через FFmpeg.
