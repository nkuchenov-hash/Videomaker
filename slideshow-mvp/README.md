# Family Slideshow Maker

Быстрое статическое веб-приложение для семейного слайдшоу.

## Что умеет

- загрузка фото и видео;
- загрузка музыки;
- авто-подгонка длительности фото под длину музыки;
- ручной порядок кадров;
- выделение сразу нескольких кадров и удаление пачкой;
- сохранение проекта локально в браузере через IndexedDB;
- открытие последнего сохранённого проекта в этом же браузере;
- ручное кадрирование X/Y;
- стартовый и финальный зум;
- форматы 16:9, 4:3, 1:1, 9:16 и свой размер;
- экспорт WebM через браузер;
- WebM дополнительно патчится длительностью, чтобы лучше проматывался в плеерах.

## Важно

Файлы не отправляются на сервер. Всё работает локально в браузере пользователя. GitHub Pages только отдаёт HTML/CSS/JS.

Сохранение проекта хранится в конкретном браузере на конкретном компьютере. Если открыть сайт на другом компьютере или в другом браузере, сохранённый проект там не появится.

Для максимально совместимого MP4/H.264 под телевизор лучше позже добавить desktop/local FFmpeg-экспорт.
