# Family Slideshow Maker

Быстрый статический веб-MVP для GitHub Pages.

## Что умеет

- загружать фото и видео;
- загружать отдельную музыку;
- менять порядок кадров стрелками или drag-and-drop;
- автоматически подгонять длительность фото под длину музыки;
- сохранять длительность видео при автоподгонке, а оставшееся время распределять между фото;
- выбирать формат/разрешение: 16:9, 4:3, 1:1, 9:16 или custom;
- настраивать кадрирование каждого фото/видео;
- отдалять фото до 0.25×, чтобы влезал весь контент;
- настраивать зум в начале и в конце кадра;
- экспортировать WebM прямо в браузере.

## Важные ограничения MVP

- Экспорт из браузера — WebM, не MP4.
- Звук из вставленных видео отключается; используется выбранная музыка.
- Экспорт идёт примерно в реальном времени: 3 минуты видео ≈ 3 минуты экспорта.
- Для стабильного MP4/H.264 под телевизор лучше следующий шаг — локальный desktop/FFmpeg экспорт.

## Как разместить на GitHub Pages

1. Загрузить файлы `index.html`, `styles.css`, `app.js`, `.nojekyll`, `README.md` в репозиторий.
2. Открыть Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /root.
6. Открыть опубликованный URL.

Если проект лежит в папке `slideshow-mvp`, URL будет вида:

```text
https://USER.github.io/REPO/slideshow-mvp/
```
