# Family Slideshow Maker

Статическое веб-приложение для быстрого семейного слайдшоу.

## Главное

- фото + видео + музыка;
- форматы 16:9, 4:3, 1:1, 9:16 и свой размер;
- список кадров слева, переключение стрелками;
- Shift+клик и Shift+стрелки для выделения диапазона;
- удаление выбранных кадров пачкой;
- сохранение проекта локально в браузере;
- ручной зум, сдвиг X/Y и режим фона;
- проверка начала/середины/конца выбранного кадра;
- MP4-пакет для финального рендера через FFmpeg.

## Экспорт

Основной экспорт — **Скачать MP4-пакет**. Он скачивает архив с проектом, медиафайлами и готовыми файлами для локального рендера:

- `project.json`;
- `assets/`;
- `render_mp4.py`;
- `run_mp4_render.bat`;
- `install_ffmpeg_windows.bat`.

Распакуй пакет и запусти `run_mp4_render.bat`. На выходе получится **MP4 / H.264 / AAC** с нормальной перемоткой.

WebM оставлен только как дополнительный быстрый черновик.

## Как обновить на GitHub Pages

Загрузи файлы из этой папки в:

```text
https://github.com/nkuchenov-hash/Videomaker/upload/main/slideshow-mvp
```

Файлы для загрузки:

```text
index.html
styles.css
app.js
README.md
.nojekyll
```

После загрузки нажми `Commit changes`, подожди 1–2 минуты и обнови сайт через `Ctrl + F5`.
